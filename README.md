# Paper Soul

**Paper Soul** é um jogo 2D de puzzle e plataforma desenvolvido em GameMaker.
O projeto apresenta fases curtas com foco em mecânicas de alternância entre estados, explorando navegação, lógica espacial e interação com o ambiente. Este jogo foi desenvolvido para a 
GameJam "NoneJam 12" do Curso de Desenvolvimento de jogos da NoneClass (https://itch.io/jam/nonejam-12)

---

O jogo é estruturado em desafios progressivos distribuídos em múltiplas fases. Cada nível introduz variações mecânicas que exigem observação, adaptação eresolução de puzzles. A progressão é baseada na conclusão das fases.

---

O progresso do jogador é salvo automaticamente. O sistema registra fases concluídas e configurações de áudio nas opções do jogo.
